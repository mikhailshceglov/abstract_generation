# Лабораторная: автоматические рефераты текстов

## Цель

По массиву исходных текстов построить массив рефератов длиной не более **300 символов** (включая пробелы). Тривиальное решение «первые 300 символов документа» не используется.

## Алгоритм

**Извлекательное** реферирование (`summarizer/summarize.py`): текст режется на предложения (`razdel`), для каждого считается важность как смесь **TextRank** (граф сходства TF‑IDF предложений + PageRank) и **лексического TF‑IDF** (вес 0,35 / 0,65). Предложения с наибольшим баллом жадно добавляются в реферат, пока не исчерпан лимит 300 символов; при переполнении выполняется обрезка по границе слова.

## Данные

| Файл | Содержимое |
|------|------------|
| `data/extractive_texts.json` | 25 исходных текстов (тема «Искусственный интеллект», ~300 символов) |
| `data/extractive_references.json` | Эталонные **экстрактивные** рефераты |
| `data/abstractive_texts.json` | Те же 25 текстов (для сравнения режимов) |
| `data/abstractive_references.json` | Эталонные **абстрактивные** рефераты (пересказ своими словами) |

Промпты для генерации эталонов нейросетью: `data/prompt_extractive.txt`, `data/prompt_abstractive.txt`.

## Метрики (ROUGE F-score, среднее по корпусу)

Один и тот же `summarize()` сравнивается с двумя наборами эталонов на **одинаковых** исходниках:

| Эталон | ROUGE-1 | ROUGE-2 | ROUGE-L |
|--------|---------|---------|---------|
| Экстрактивный | 0,9141 | 0,8185 | 0,7445 |
| Абстрактивный | 0,3986 | 0,1209 | 0,3177 |

Извлекательный алгоритм согласуется с экстрактивным эталоном сильнее, чем с абстрактивным — ожидаемо при метриках на совпадении n-грамм.

Визуализация: `data/rouge_comparison.png` (скрипт `scripts/plot_rouge_comparison.py`).

## Запуск

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

Оценка:

```bash
.venv/bin/python evaluate.py
# экстрактивный эталон по умолчанию (extractive_*.json)

.venv/bin/python evaluate.py --texts data/abstractive_texts.json --references data/abstractive_references.json
```

График:

```bash
.venv/bin/python scripts/plot_rouge_comparison.py
```

Тесты:

```bash
.venv/bin/python -m pytest
```

## Структура репозитория

- `summarizer/` — модуль `summarize()`
- `evaluate.py` — средние ROUGE между предсказанием и эталоном (`rouge-score`, токенизация `razdel`)
- `tests/` — проверки длины, нетривиальности, согласованности данных
